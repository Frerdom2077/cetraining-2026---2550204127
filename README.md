##功能介绍
核心功能：
 从文件导入学生成绩数据（学号、姓名、多科成绩）
 按学号或姓名查询学生成绩
 计算每个学生的总分与平均分
 按平均分排序并输出排名
扩展功能：
 支持新增、删除、修改学生成绩
 将统计结果导出为新文件
涉及知识：结构体/类、数组/链表、文件读写、排序、查找

一、代码结构概览

这份代码是一个学生成绩查询与统计工具，包含三个核心部分：

Student 类：存储单个学生的学号、姓名、五科成绩，自动计算总分和平均分

GradeManager 类：管理所有学生数据的增删改查、文件读写、排序统计

interactive_menu 函数：提供命令行交互菜单，用户通过输入数字选择功能

代码框架设计：AI根据需求描述生成面向对象的类结构设计，包括Student类、GradeManager类、交互菜单的框架
方法实现：AI生成各方法的实现代码，如文件读写、数据解析、排序算法、统计计算等

二、使用AI过程中遇到的问题
问题一：AI生成的代码存在语法错误
具体表现：缩进错误、括号不匹配、变量名拼写错误
修正方法：在PyCharm中运行代码，根据报错信息定位错误行，手动修正语法问题
问题二：AI生成的逻辑不符合实际需求
具体表现：文件路径处理不当、数据格式解析错误、边界条件未考虑
修正方法：实际运行测试用例，对比预期输出和实际输出，针对性修改逻辑
问题三：AI生成的代码缺少异常处理
具体表现：文件不存在时崩溃、输入非数字时崩溃、空列表操作报错
修正方法：增加try-except块、增加空值判断、增加类型检查
问题四：AI生成的中文显示存在乱码
具体表现：控制台输出中文为乱码、CSV文件中文显示异常
修正方法：确保文件编码为UTF-8、增加编码声明、Windows系统执行chcp 65001
问题五：AI生成的代码性能不佳
具体表现：大数据量时查询缓慢、排序效率低

修正方法：将线性查找改为字典索引、使用数据库替代文件存储
